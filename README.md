# Winter Minimap

![Winter Map](/images/2_1.webp)

## Description
This mod replaces default minimap in NFSW with modded, like it's snowy.

### Installation
1. Download the mod files.
2. Extract "Tracks" folder to ...\Game Files.
3. Enjoy the mod!\
Note 1: Make sure to back up your original minimap before installing mod.\
Note 2: **Online WorldUnited.gg server WILL NOT WORK**, due the custom map. On any other online/offline (include WUGG Offline) servers will work.\
Note 3: Maybe I will make a HD version for bigger HUDs.
